# Analysis Chain Versions

This private version registry records the local paper-analysis chain variants used in controlled runs.

- `base_v0_20260515.md`: pre-optimization/base chain used as the old comparison baseline.
- `analysis_chain_v1_20260522.md`: current optimized chain after the 8-paper image-standard validation run.

Quality policy: a newer chain becomes default only after structural checks pass and manual/LLM review shows no quality regression on shared papers.
