# ResearchFlow Paper Index

This directory is generated by `papers-build-index` from `obsidian-vault/paper_list.csv` and `obsidian-vault/analysis/` frontmatter.

## Entry Points

- `obsidian-vault/index/index.jsonl`
- `obsidian-vault/index/paper_index.md`
- `obsidian-vault/index/_AllPapers.md`
- `obsidian-vault/index/by_topic/`
- `obsidian-vault/index/by_method/`
- `obsidian-vault/index/by_dataset/`
- `obsidian-vault/index/by_venue/`
- `obsidian-vault/index/by_year/`

Topic pages use coarse top-level research areas. Method pages use normalized
method families; exact method names remain in `index.jsonl`.

Generated index files are local research artifacts and are ignored by default.
